## Key Interaction Points

### 1. Persona Switching
- **Experienced RM**: Concise, technical, assumes knowledge
- **New RM**: Detailed, explanatory, step-by-step guidance
- **Specialist RM**: Deep technical detail (not shown in demo)

### 2. Compliance Flagging
- Automatic detection of conflicting clauses
- Visual alert banner
- Actionable recommendations
- Escalation guidance

### 3. Citation Tracking
- Every response includes source references
- Clickable citation chips
- Section numbers for easy lookup

### 4. Feedback Loop
- Thumbs up/down on every response
- Tracks satisfaction in real-time
- Feeds into evaluation metrics

### 5. Multi-lingual Support
- Italian language responses
- Maintains professional tone
- Context-aware translations
